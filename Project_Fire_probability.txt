Имеется набор с районами города. В каждом районе есть набор адресов
домов различной этажности.
Адреса имеют координаты.
Есть пожарные части. Есть координаты этих пожарных частей.
Пожарные части закреплены за районами (районы для разных частей могут совпадать)
У пожарных частей - различная комплектация оборудования для тужения пожаров
(пожарные автомобили; большая ёмкость для воды; мобильная насосная станция;
возможность тушения хим. реактивами; лестница для эвакуации различной этажности, и т. д.)
Имеются категории пожаров.

1. Определяется ближайшая (по радиусу) пожарная часть для выезда на пожар.
2*. После определения категории пожара, происходит привлечение техники
других пожарных частей, в зависимости от назначенной категории.
3. По имеющейся суточной статистике пожаров в городе определяется вероятность возникновения
пожара для каждого из районов, которая влияет на привлечение техники пожарных частей
эитих районов.

* - пожарная часть привлекается, если вероятность возникновения нового пожара,
в закрепленных за ней районах, по имеющейся суточной статистике - ниже 75%
